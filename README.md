# 🌸 Fahreza Bio — Personal Link in Bio

> A beautifully crafted Japanese sakura-themed bio link page, built with love by **Muhammad Fahreza**.

🔗 **Live Demo:** [fahreza-bio.netlify.app](https://fahreza-bio.netlify.app/)

---

## ✨ What is this?

A personal **Link in Bio** website — one page, all your important links, with a premium feel. Inspired by Japanese cherry blossom (桜 / Sakura) aesthetics, complete with falling petals, ambient soundscapes, and smooth animations that make it feel more like an *experience* than a profile page.

---

## 🎴 Features

| Feature | Description |
|---|---|
| 🌸 **Sakura Background** | Animated falling petals rendered on HTML5 Canvas |
| 🎵 **Ambient Audio** | Generative sakura soundscape with wind, chimes & birds via Web Audio API |
| ⚡ **Smooth Scrolling** | Powered by Lenis for buttery-smooth scroll behavior |
| 🖱️ **Custom Cursor** | Pink glowing cursor with trail effect (desktop only) |
| 🎬 **Animated Preloader** | Custom SVG sakura scene with progress bar |
| 🧲 **Magnetic Buttons** | Social buttons react to mouse proximity |
| 🎮 **Konami Code Easter Egg** | Hidden secret for the curious ones (↑↑↓↓←→←→BA) |
| 📱 **Swipe Detection** | Touch-friendly swipe notifier for mobile |
| 🔠 **Split Text Reveal** | Character-by-character name animation on load |
| ♿ **Reduced Motion** | Fully respects `prefers-reduced-motion` |
| 🔍 **SEO Ready** | Dynamic meta tags + Open Graph + Twitter Card |

---

## 🛠️ Tech Stack

```
React (Vite)  ──  UI framework + lightning-fast build tool
GSAP          ──  Animations, ScrollTrigger, magnetic effects
Lenis         ──  Smooth scroll library
Web Audio API ──  Generative ambient audio (no audio files needed!)
HTML5 Canvas  ──  Real-time sakura petal & wind rendering
CSS3          ──  Glassmorphism, custom properties, keyframe animations
Netlify       ──  Deployment & hosting
```

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/MhmdFahreza/Bio.git
cd Bio

# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
